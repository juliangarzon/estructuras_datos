ADT vs Implementación con Java

ADT = Interface en Java

¿QUÉ hace?

  • • Define las operaciones (métodos)
  • • Especifica el comportamiento esperado
  • • NO dice CÓMO implementar
Ejemplo:
interface Lista { add(); remove(); get(); }

Implementación = Clase en Java

¿CÓMO lo hace?

  • • Código específico (implements)
  • • Variables de instancia
  • • Algoritmos concretos
Ejemplo:
class ArrayList implements Lista

🎯 La magia de Java:

Una interface puede tener múltiples implementaciones

Lista puede ser ArrayList, LinkedList, Vector... ¡Todas diferentes por dentro!

¿Por qué son importantes?

❌ Sin estructuras adecuadas:

  • • Programas lentos
  • • Código difícil de mantener
  • • Uso excesivo de memoria
  • • Bugs difíciles de encontrar

✅ Con estructuras correctas:

  • • Rendimiento óptimo
  • • Código elegante y claro
  • • Uso eficiente de recursos
  • • Fácil de extender

💡 Ejemplo Real

Instagram: Buscar entre 1 billón de fotos en menos de 100ms
Netflix: Recomendar películas a 200M usuarios simultáneamente
WhatsApp: Entregar 65 billones de mensajes al día

ADT con UML: Interface Animal

El ADT (Interface) define QUÉ debe hacer todo animal:

<<interface>>
Animal
+ comer() : void
+ dormir() : void
+ hacerSonido() : void
+ moverse() : void
+ obtenerEspecie() : String

🎯 ¿Qué especifica esta interface?

Operaciones comunes:

  • Todo animal debe comer
  • Todo animal debe dormir
  • Todo animal hace algún sonido

Pero NO dice CÓMO:

  • ¿Qué come cada animal?
  • ¿Cómo se mueve?
  • ¿Qué sonido hace?

Implementaciones: Conejo y Gato

Cada clase implementa el ADT de manera específica:

Conejo
implements Animal
Atributos:
- colorPelaje: String
- longitudOrejas: double

Métodos del ADT:
+ comer(): "Como zanahorias"
+ dormir(): "Duermo en madriguera"
+ hacerSonido(): "No hago sonido"
+ moverse(): "Salto"
+ obtenerEspecie(): "Conejo"

Métodos únicos:
+ cavar(): void
+ moverOrejas(): void
Gato
implements Animal
Atributos:
- raza: String
- vidas: int

Métodos del ADT:
+ comer(): "Como pescado"
+ dormir(): "Duermo 16 horas"
+ hacerSonido(): "Miau"
+ moverse(): "Camino sigilosamente"
+ obtenerEspecie(): "Gato"

Métodos únicos:
+ ronronear(): void
+ trepar(): void
+ cazarRaton(): boolean

🎯 ¡La magia del polimorfismo!

Animal miMascota = new Gato();
miMascota.hacerSonido(); // "Miau"
